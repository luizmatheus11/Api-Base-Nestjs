## Installation

```bash
$ npm install

$ docker compose up

# setup Prisma
$ npx prisma generate

$ npx prisma db push
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Documentation

